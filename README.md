# DevelStrap VueJS

Complete Vue web app. Quickly straps all the needs for a complete Vue web app solution.

See [DEMO](http://develstrapvue.net/)

## Install

`npm install`

## Build

### Development

`npm run dev`

### Production

`npm run dist`
